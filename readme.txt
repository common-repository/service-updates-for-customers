== Description ==

SUC (services updates for customers) is a plugin which auto update the subscriber for service updates through email. The customer first subscribe for your services and then he/she should be notify if admin post new comments on any service. 
you can check the <a href="http://demo.cybersourcepk.com">Demo here </a>


Features:

== Installation ==


1. Upload the folder ` service-updates-for-customers` to the `/wp-  
   content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place short code ` [services_updates]` in page where you want to display service.

== Frequently Asked Questions ==

= How to get subscriber for services updates =
  You have to use the widget to display the subscription form.

= Can I disable some users from updates =
   Yes you can, under setting click on Service Updates Via Email then check   
   the setting.
